This folder contains a collection of EEG data cleaning functions written by Christian Kothe (the clean_***.m functions), as well as an EEGLAB plugin and GUI (pop_clean_rawdata) by Makoto Miyakoshi.
The other two readme files are part of the original data cleaning function collection (included for reference).
